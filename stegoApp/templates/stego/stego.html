What is Stego?

Steganography is the art of hiding data in plain sight. There are several different techniques and methods used to accomplish this task, however this tool is focused on using the LSB (least significant bit) technique to hide data in the RGB(A) values of an image. This is accomplished by converting each pixel of an image into it's RGB(A) values and modifying the least significant bit (the most left bit in each byte for each value in a pixel). This technique is somewhat simple to accomplish and as the least significant bit is the bit modified, detection from simply viewing the image is not possible under reasonable circumstances.
